<h1 align="center">Проект: Место</h1>

---

_**"Mesto"** - это интерактивная страница, на которой можно добавлять и удалять фотографии, а также ставить лайки._

---

## Используемые технологии

1. HTML
2. CSS
3. Flexbox
4. Grid Layout
5. Адаптивная верстка
6. Javascript
